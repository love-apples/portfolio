Главная
Мои проекты (пропущено ~20 парсеров):
  • Botnet "Александрия" (C#) Мой первый простейший ботнет. Возможно вошёл бы в номинацию "Худший ботнет 2021".

    Проект доступен для просмотра исходного кода. Использование только в ознакомительных целях.
  • Discord Utility (Python) Один из крупных проектов. Позволяет проверить на доступность токены Discord, а также сделать рассылку по всем каналам.

    Проект не доступен для просмотра исходного кода.
  • GD Platformer Mode (Python) Проект, написанный совместно с nekitdev (GitHub его). Мод позволяет управлять игроком с помощью стрелок на клавиатуре.

    Проект доступен для просмотра исходного кода.
  • Discord Checker (C#) Desktop софт для проверки на доступность токенов Discord. Впрочем то же самое что и Discord Utility, только без рассылки.

    Проект доступен для просмотра исходного кода. Использование только в ознакомительных целях.
  • VK Dumper (Python) Скрипт на коленке, позволяющий собрать историю фотографий из переписок людей полученных из метода VK "messages.getConversations". Результат сохраняется в папку "response".

    Проект доступен для просмотра исходного кода. Использование только в ознакомительных целях.
  • Piar Bot (Python) Вообще не помню что делал это. Нашёл в своих проектах. Стиль кода мой. Видимо предназначен для продажи рекламы, где её текст будет распространяться по ВКонтакте. Возможно в будущем камбекну этот проект.

    Проект не доступен для просмотра исходного кода.
  • Kwork listener (Python) Скрипт прослушивающий биржу Kwork`а. Уведомления о новых проектах отправляются через Telegram бот.

    Проект доступен для просмотра исходного кода.
  • RCGD Bot (Python) Был бот по игре Geometry Dash. Исходники утеряны.
Список будет пополняться.
